# 公司简介
*卜博教育科技公司是一个以快乐治学为宗旨的教育科技公司，公司创办于国际化大都市中国上海。卜博教育科技公司是全海归团队，定位于帮助学生提高知识技能、实现人生梦想的教育资源平台，为学生提供经过我们严格筛选后的优质国际教育资源，提供一站式的教育服务。*
# 企业愿景
一个以快乐治学为宗旨的教育科技公司
# 企业使命
利用科技技术，辅助传统教育行业，培养创新精神，帮助学生提高知识技能，实现人生梦想，提供一站式的国际教育资源服务
